Curso NodeJS

Criando seu primeiro projeto NodeJS
npm init ou npm init -y
Teclar enter confirmando todas informações, as configurações estarão no arquivo package.json
        Crie um arquivo index.js
            let nome = 'Wagner';
            let idade = 42;
            
            console.log(`Meu nomé é ${nome} e tenho ${idade} anos`);
Inicia projeto
node index.js
Arquivo App.tsc dentro da src

Instalando typescript

npm install -g typescript
Iniciar arquivo de configuração do typescript
tsc --init (cria arquivo tsconfig.json)
Configurando Typescript 
        Abra o arquivo tsconfig.json e altere
        "target": "es6",
        Adicione a linha
        "moduleResolution: "node",
        Descomente definindo caminho
        "outDir": "./dist",
        "rootDir": "./src",
Instalar dependência typesnode
npm install --save-dev @types/node

Executar código da dist
tsc -w (monitora as modificações)

Agora para visualizar novamente
node ./dist/index.js
Personalizar configurações no arquivo package.json 
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
            "start": "node dist/index.js",
            "watch-ts": "tsc -w"
          },
Para confirmar execute
npm run watch-ts inicar monitoramento
npm run start iniciar projeto
